# g4-example-advanced-dna


The `moleculardna` sub-example demonstrates Geant4's full radiation-biology pipeline: track-structure physics down to a few eV, a chemistry stage propagating reactive radicals (OH·, H·, e⁻aq) via the IRT model, and DNA damage scoring (SSB/DSB) against fractal-placed genomes of billions of base pairs. It proves that Geant4 is a multi-stage physics-chemistry-biology simulator and not merely an HEP/medical-physics tracker — and simultaneously marks the outer bound of what a generic GDML+TTree main can reasonably attempt.

## Key concepts demonstrated
- `track-structure-physics` — `G4EmDNAPhysics_option2/4/6` track every electron individually down to ~7.4 eV in liquid water with no condensed-history approximations; step ≈ atomic spacing
- `chemistry-stage` — Geant4 has a post-physics chemistry kernel that propagates reactive species with IRT timestep models; activated by `/process/chem/TimeStepModel IRT_syn` and bounded by `/scheduler/endTime`
- `fractal-dna-placement` — billions of base-pair positions come from external fractal-definition text files (Hilbert curves, prism scaffolds) generated by the `fractaldna` Python tool; not from C++
- `sdd-output-format` — Standard for DNA Damage format; simulation output feeds independent repair/survival model code downstream, cleanly separating physics from biology
- `voxel-optimization` — `/dnageom/setSmartVoxels 1` is required to make navigation tractable for 200 000+ placed fragments; without it the simulation runs out of memory
