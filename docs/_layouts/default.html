Claude Code plugin · MIT

Describe a detector. Get a simulation.

geant4_claude turns plain-English physics specs into validated GDML, runs them in a pinned apptainer image, and analyzes the ROOT output on the host with uproot — no host-side Geant4, no compiled code shipped, no surprises.

Install in two lines.

Add the marketplace, install the plugin. Apptainer ≥ 1.4 must already be on the host; the plugin pulls its pinned container image on first use. /plugin update handles upgrades.

claude-code · plugins 
› /plugin marketplace add zhaozhiwen/geant4_claude
› /plugin install geant4-claude
